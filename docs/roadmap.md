# Implementation Roadmap

## Phase 1 — MVP (2-4 weeks)

- Job creation API
- Script generation worker
- Scene breakdown worker
- Voice generation worker
- FFmpeg assembly with static visuals
- Manual YouTube upload trigger

**Goal:** Generate a complete video with minimal human intervention.

## Phase 2 — Automation (3-5 weeks)

- Visual generation per scene (DALL·E/Leonardo)
- Thumbnail + SEO worker
- YouTube OAuth + auto upload
- Scheduler for daily posting

**Goal:** One-click automated publishing.

## Phase 3 — Optimization (ongoing)

- A/B testing for hook/title/thumbnail
- Analytics feedback loop for prompt tuning
- Cost optimization (provider routing)
- Multi-language expansion

**Goal:** Improve retention, CTR, and unit economics.

## Suggested milestones

1. **M1:** Job lifecycle and observability.
2. **M2:** Voice + assembly quality baseline.
3. **M3:** Auto-upload reliability and retries.
4. **M4:** Analytics dashboard and optimization loop.
